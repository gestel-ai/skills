# Meta And Instagram Static Fit

This reference captures GESTEL's local MVP placement contract. Platform policies and specs can change, so verify live documentation or add a dated source before making freshness-sensitive claims.

## Supported Ratios

| Ratio | Use |
| --- | --- |
| `4:5` | Default first-pack feed image |
| `1:1` | Square feed expansion after candidate selection |
| `9:16` | Story/Reels static expansion after candidate selection |

The first pack defaults to `4:5`. Do not generate all ratios before the user selects candidates unless the user explicitly changes the scope.

## Static Review Checklist

- Product remains recognizable on mobile.
- Headline, badge, CTA, logo, and price text stay inside the safe visual area for the requested ratio.
- Text is short enough to read at feed size.
- CTA is clear but not deceptive.
- Required disclosure is visible when needed.
- Korean copy sounds natural and commerce-specific.
- Visual treatment does not imply video, motion, before/after transformation, or product functionality that is not present.

## Platform Scope

This active skill covers Meta and Instagram static ads only. TikTok hooks, Naver, Kakao, Coupang, Google, and video learnings may inform later work, but they are not active output contracts for the MVP static review gate.
